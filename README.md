I have developed an AI Joke Generator that generates random funny jokes when the user clicks on the “Get Joke” button. The application is built using HTML, CSS, and JavaScript, where JavaScript handles the main functionality. When the button is clicked, an API request is sent to a joke generator service, which returns a random joke in JSON format. This joke is then displayed on the webpage using DOM manipulation. The project demonstrates concepts like event handling, API integration, and asynchronous programming. It is simple, interactive, and provides a new joke on every click, making it user-friendly and engaging.
